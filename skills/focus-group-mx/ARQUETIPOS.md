## Por qué este balance

1. **Refleja la pirámide MX real**: ~10% A/B+C+, ~17% C, ~36% C-+D+, ~37% D+E. Tenemos 4+2+2+4+2 = 14 con peso similar.
2. **No es solo "consumidor masivo"**: incluye E (jornaleros) y A/B (top 5%) para detectar cuándo un ad solo resuena con un slice estrecho.
3. **Mezcla urbano denso (CDMX) con periferia (Ecatepec/Chimalhuacán) y rural (Chiapas)**.
4. **Cubre arquetipos psicográficos distintos**, no solo demográficos.

## Lo que NO está

- **Migrantes de retorno** (deportados, repatriados) → para v2
- **LGBT explícito** → para v2
- **Indígenas con lengua materna no-español** → para v2 (Don Tomás puede serlo implícito)
- **Discapacidad** → para v2
- **Migración interna activa** (recién llegados a otra ciudad) → para v2

Estos son ejes válidos pero entrar en v1 con 14 ya cubre el grueso del mercado consumo MX.
